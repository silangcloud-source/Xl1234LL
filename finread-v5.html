Notion 同步

将卡组、文章归档、复习进度同步到 Notion 页面。跨设备访问，永久保存。

未连接
⚠️ Notion API 不支持浏览器直连 (CORS 限制)。需使用 CORS 代理或部署 Cloudflare Worker。详见「说明」标签页。
配置 Token 后启用同步
三种使用方式
① 推荐 · CORS 代理：在配置中填入公开代理 (如 https://corsproxy.io/?)，可直接调用 Notion API。

② 推荐 · 自部署 Cloudflare Worker：5 分钟部署一个代理 Worker，最安全。模板见下方。

③ 兜底 · 手动粘贴：在「同步」页点击「复制 JSON」，在 Notion 创建一个 Code Block 粘贴。
1. 访问 https://www.notion.so/profile/integrations
2. New integration → 类型选 Internal → 复制 Token (ntn_...)
3. 在你的 Notion hub 页面 → Share → 添加该 Integration → 授予 Edit 权限
4. 把 Token 粘贴到上方「Integration Token」
部署到 Cloudflare Workers (免费) 后，填入「CORS 代理 URL」框，例如 https://your-worker.username.workers.dev/v1，最后的 /v1 必须有。
