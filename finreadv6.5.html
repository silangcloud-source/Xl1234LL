☁ JSONBin 同步

免费云端同步，无需服务器。词卡 + 复习日志跨设备实时共享。

未连接
📌 获取 API Key：注册 jsonbin.io → 右上角 API Keys → 复制 Master Key（以 $2a$ 开头）

📋 使用步骤

  1. 访问 jsonbin.io 免费注册
  2. 右上角 → API Keys → 复制 Master Key
  3. 粘贴到「配置」→ 保存
  4. 点「⬆ 推送」完成首次同步，Bin ID 自动填入
  5. 换台设备：填入同一 Master Key + Bin ID → 点「⬇ 拉取」

⚠ 注意

  • 免费套餐限 100 KB/次，超出时词卡仍会同步，文章正文会被省略
  • wordHistory（例句缓存）不同步，可在设备上重新生成
  • 推送是全量覆盖，拉取是智能合并（不会丢失本地更新）

☁ WebDAV 同步

通过 WebDAV 协议同步词卡到坚果云、NextCloud 等私有存储，数据完全在你掌控之中。

未连接
📌 坚果云应用密码：登录 jianguoyun.com → 账号信息 → 安全选项 → 添加应用密码

📋 坚果云设置步骤

  1. 访问 jianguoyun.com 注册/登录
  2. 右上角头像 → 账号信息 → 安全选项
  3. 「第三方应用管理」→ 添加应用密码，记录生成的密码
  4. 回到此页「配置」Tab，填入：
    服务器：https://dav.jianguoyun.com/dav/
    用户名：坚果云账号邮箱
    应用密码：刚才生成的密码
  5. 点「保存」→ 切换到「同步」→「⬆ 推送」完成首次同步
  6. 换台设备填入同一配置 → 点「⬇ 拉取」即可

⚠ 注意

  • 推送是全量覆盖，拉取是智能合并（不丢失本地进度）
  • 坚果云免费版 1 GB 存储，词卡文件约 50–200 KB，完全够用
  • 应用密码存于本地 localStorage，仅限本设备使用
  • 同样支持 NextCloud、Alist、群晖等兼容 WebDAV 的服务

引擎配置

学习数据优先存于本地 IndexedDB，localStorage 作为兼容兜底。云同步：J (JSONBin) 或 W (WebDAV/坚果云) 按钮。
